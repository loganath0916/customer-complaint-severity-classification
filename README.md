# Customer Complaint Severity Classification using NLP

## Overview
Developed an NLP-based classification system to automatically categorize customer complaints into **Low, Medium, and High Priority** levels, enabling faster issue resolution and efficient support ticket prioritization.

## Key Features
- Text preprocessing using NLTK (Tokenization, Stopword Removal, Lemmatization)
- Feature extraction using CountVectorizer, TF-IDF, and Word2Vec
- Feature importance analysis using Random Forest
- Multi-class classification using Logistic Regression, Random Forest, SVM, XGBoost, and LSTM
- Streamlit-based web application for real-time complaint severity prediction

## Tech Stack
**Python, NLP, NLTK, Scikit-learn, TensorFlow, Keras, XGBoost, Streamlit, Pandas, NumPy**
